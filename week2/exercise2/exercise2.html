CSYM019 - Internet Programming

Lecturer: Tom Butler

This subject focuses on developing client and server-side software, as well as web applications for usage on the World Wide Web. It focuses on the technologies that are used to create, implement, and deploy such software.

CSYM015 - Intelligent Systems

Lecturer: Scott Turner

The goal of this course is to educate students the core theory and applications of search algorithms and agents. The fundamental concepts will be presented, followed by examples of how intelligent systems are applied in Internet applications.
